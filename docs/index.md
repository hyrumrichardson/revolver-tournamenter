## Description

**Welcome!** This is a Python script for generating pairings and a scoresheet for the monthly Revolver League Magic: The Gathering constructed tournaments that take place at Austin Books & Games (formerly Outlaw Moon).

## Quickstart

```sh
git clone https://github.com/hyrumrichardson/revolver-tournamenter
cd revolver-tournamenter

pip install -e .
revolver-tournamenter --filename input.txt (--rounds 5)
```

where input.txt looks like

```sh
Player A
Player B
Player C
...
```
